## Arrays vs ArrayList

| Feature | Array | ArrayList |
|---------|-------|-----------|
| Size | Fixed | Dynamic |
| Syntax | `int[]` | `ArrayList<Integer>` |
| Performance | Faster | Slightly slower |
| Flexibility | Limited | Many methods |
