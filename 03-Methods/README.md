# Module 3: Methods

Methods are reusable blocks of code that perform specific tasks. They help organize code, avoid repetition, and make programs more maintainable.

## Why Methods Matter

**DRY Principle**: Don't Repeat Yourself
- Write code once, use it many times
- Changes only need to be made in one place
- Easier to test and debug
- Makes code more readable

## Real-World Applications

- **Banking**: `calculateInterest()`, `validatePassword()`
- **Gaming**: `calculateDamage()`, `checkCollision()`
- **E-commerce**: `calculateTax()`, `applyDiscount()`
