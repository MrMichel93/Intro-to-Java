- **Game Development**: Minecraft is written in Java!

### Career Paths
- Junior Java Developer ($60K-$80K entry level)

## ğŸ› ï¸ Getting Started

### Prerequisites
- A computer (Windows, Mac, or Linux)
- Internet connection for initial setup
- Text editor (we'll set up a proper IDE together)
- Willingness to learn and make mistakes!

### Setup Steps
1. Go to [Module 0: Getting Started](00-Getting-Started/)
2. Follow the installation guide for your operating system
3. Set up your IDE (VS Code or IntelliJ IDEA)
4. Write and run your first Java program
5. If you get stuck, check the troubleshooting guide

### Quick Start (Command Line)
```bash
# Check if Java is installed
java -version

# If not installed, download from:
# https://www.oracle.com/java/technologies/downloads/
# or https://aws.amazon.com/corretto/ (Amazon Corretto 17)

# After installation, verify:
javac -version  # Should show 17.x.x

# Create your first program
echo 'public class Hello { public static void main(String[] args) { System.out.println("Hello, Java!"); } }' > Hello.java

# Compile
javac Hello.java

# Run
java Hello
```

## ğŸ’¡ Learning Tips

**For Success:**
- **Code Every Day**: Even 30 minutes is better than cramming
- **Type, Don't Copy-Paste**: Muscle memory matters in programming
- **Break Things**: Experiment! See what happens when you change code
- **Read Error Messages**: They're your friends, not enemies
- **Google is Your Friend**: Every programmer searches syntax
- **Join a Community**: Don't learn alone (see resources below)
- **Build Projects**: Apply what you learn immediately
- **Teach Others**: Best way to solidify understanding

**When You're Stuck:**
1. Read the error message carefully
2. Check the troubleshooting guide
3. Use print statements to debug
4. Take a break and come back fresh
5. Ask for help in the community
6. Remember: Struggling is learning!

**Mindset:**
> "Programming is a skill you build, not a talent you're born with. Every expert was once a beginner. Every error is a step toward mastery."
